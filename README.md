# COLSI | Cost of Living Strategic Intelligence System

COLSI is a high-fidelity, data-driven analytical platform designed to monitor, analyze, and forecast regional economic stress factors. Built with a focus on precision and executive-level visibility, the system provides a unified interface for tracking cost-of-living variances across energy, food, transport, and housing sectors.

## Strategic Overview

The system utilizes advanced normalization algorithms to synthesize complex economic data into a standardized **Strategic Cost Index (1-100)**. This allows decision-makers to identify baseline deviations and regional anomalies with immediate clarity.

### Primary Intelligence Pillars

*   **⚡ Energy Infrastructure**: Real-time monitoring of utility costs, grid load variance, and input price volatility.
*   **🛒 Food Security Index**: Granular tracking of core commodity prices and supply chain reliability metrics.
*   **🚚 Logistics & Mobility**: Benchmarking fuel price impacts against sector-wide transportation overhead.
*   **🏢 Residential Stability**: Long-term variance analysis of urban housing costs and market availability.

## System Architecture

COLSI is engineered for performance and scalability, leveraging a modern technological framework:

*   **Frontend**: React 18 with TypeScript for robust type safety.
*   **Styling**: High-fidelity vanilla CSS architected for fluid, responsive, and premium visual experiences.
*   **Animations**: Framer Motion and GSAP for interactive micro-animations and state transitions.
*   **Analytics**: Custom scoring engines and predictive ML adapters (simulated for strategic briefing).
*   **Reporting**: Integrated PDF generation for executive performance summaries.

## Getting Started

### Prerequisites

*   **Node.js**: v18.0.0 or higher
*   **Package Manager**: npm, yarn, or pnpm

### Installation & Deployment

1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/imamdoula004/COLSI-Vite-Intelligent-System.git
    cd colsi-vite
    ```

2.  **Initialize Dependencies**:
    ```bash
    npm install
    ```

3.  **Launch the Analytics Dashboard**:
    ```bash
    npm run dev
    ```

## Scoring Methodology

The COLSI Index is calculated using a weighted harmonic mean of normalized sector indicators. All index scores are mapped to a **1-100 scale**:

*   **01 - 30 (Stable)**: Economic parameters are within historical norms.
*   **31 - 60 (Moderate)**: Observable variance requiring strategic attention.
*   **61 - 85 (High)**: Significant stress impacting regional affordability.
*   **86 - 100 (Critical)**: Immediate structural intervention suggested.
